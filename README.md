# Lazer_Termogramm_Monitor

Проект **Termogramm_My** — Python‑пакет и ноутбуки для обработки, визуализации и анализа температурных данных (термограмм).  
Содержит набор скриптов для чтения данных, построения температурных карт/графиков, а также утилиты и примеры.

## Содержание
- `Data/` — примеры исходных данных.
- `Main_process/` — основной код анализа и построения термограмм.
- `Utilities/` — вспомогательные модули (загрузка, предобработка и т.д.).
- `Debugging_Information/Bests_Models/` — сохранённые модели/результаты (если применимо).

## Возможности
- Загрузка и предобработка температурных данных.
- Построение термограмм и графиков.
- Анализ временных рядов температуры.
- Настраиваемые визуализации для различных датасетов.

## Установка и окружение

Установка:
```bash
git clone https://github.com/gvo104/Termogramm_My.git
cd Termogramm_My
```

Окружение:
```bash
pip install -r requirements.txt
```
